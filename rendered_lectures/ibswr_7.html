Question: should the lines all have the same slope?

Interactions

If x1x1 is continuous and x2x2 categorical with (K+1)(K+1) categories, in the regression

y=β0+β1x1+β2x2+ϵy=β0+β1x1+β2x2+ϵ

The predicted values are

yˆ={β0+β1x1β0+β1x1+β2,kk=0k>0y^={β0+β1x1k=0β0+β1x1+β2,kk>0

  • kk is the category of x2x2, changes the intercept

To change the slope, we add the interaction term

y=β0+β1x1+β2x2+β3x1x2+ϵy=β0+β1x1+β2x2+β3x1x2+ϵ

Question: What are the predicted values?

Answer:

yˆ={β0+β1x1β0+β1x1+β2,k+β3,kx1k=0k>0y^={β0+β1x1k=0β0+β1x1+β2,k+β3,kx1k>0

Bayesian multiple regression

Stan

For the Bayesian version with brms we can easily use the same syntax

brmfit_diamonds <- brm(log_Price ~ log_Carats*(Color + Clarity), diamond_data) # run the model

Note that this version has

  • default priors and identity (not log) link for σσ
  • default chain lengths (2000 iterations)

Summary

For multiple linear regression

y=β0+β1x1+β2x2+…+ϵy=β0+β1x1+β2x2+…+ϵ

  • the predicted value is the linear combination of the terms

Allows adjustment for confounding factors

We may need to transform continuous variables

  • for example log transformation, or log-linkage

With brms can again easily run Bayesian multiple regression

  • with the same regression formula syntax bf(… ~ …)

We can also easily include interaction between variables

Model checking (and prior definitions!) more involved
